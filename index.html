Timeline

Timeline is an easy and beautiful way to chronicle history. It has many features including autoplay, early preview, custom timers, and event read indicators. It has no dependencies but works best when used with a responsive grid-layout framework.

Getting Started

Getting started is easy. First include the timeline stylesheets and javascript. Timeline requires that you use specific ids and classes in order for it to work. Follow the basic timeline example below.

First, include stylesheets and script into your HTML

<head>
  <link href="/styles/timeline.min.css" rel="stylesheet">
</head>
<body>
  <!-- Add your Timeline and events -->
  <script src="/js/timeline.min.js"></script>
</body>

Next, add the Timeline markup and your events

<div id="tL-body" class="dark-theme round-theme"> <!-- both classes and the id are necessary -->
  <div id="tL">
    <div class="tL line"></div>
    <div class="tL ball"></div>
    <div class="tL event" data-side="right" data-timer="8000">
      <h4 class="tL title">My first event</h4>
      <time class="tL date">Today</time>
      <div class="tL body">
       This is timeline!
      </div>
    </div>
  </div>
</div>

div class="col-lg-12 text-center dark-theme round-theme" id="tL-body">

Added Auto Play to Timeline

April 16, 2015

See it in action by clicking the play circle at the top of this Timeline. It allows for variable speed with a data-timer attribute, and has a status bar to indicate when it will switch to the next event.

Removed jQuery dependency for Timeline

April 16, 2015

The jQuery library is a bit overkill to power Timeline, so I removed it and coded it up as vanilla JavaScript instead. Firefox is a bit buggy due to some document.body.scrollTop limitation.
